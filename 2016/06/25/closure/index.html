闭包

标签（空格分隔）： python

恩 听起来很厉害的样子喔
概念：内部函数对外部变量的引用

def count():
    fs = []
    for i in range(1, 4):
        def f():
             return i*i
        fs.append(f)
    return fs

f1, f2, f3 = count()

其实 感觉也不是很困难的样子哎 ╮(╯_╰)╭
闭包像个空心球 ， 你知道内部和外部是什么，但不知道中间是什么
闭包三个要素：

  • 必须有内部函数
  • 内嵌函数需要引用外部函数的变量
  • 外部函数必须返回内嵌函数

函数运行后不会被撤销。WHY
因为闭包的作用就是 在程序返回后没有被释放资源的栈区。
在多人开发时，可以避免重复定义相关变量。
